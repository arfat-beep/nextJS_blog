# Blog Website

# [Live website](https://nextjs-blog-arfat-beep.vercel.app/)

## this blog website created with GraphCMS for backend

### Dependency

- NextJs
- GraphCMS
- TailWind
- HTML react parser

### Features

- Easy conent manage
- Comment check and approve system
- Rich Text box for writing blog
- Show all comments after post
